⚡ Built for autonomous agents

Analytics for agents that run on their own.

AgentSig is a telemetry platform where any autonomous agent — bots, scrapers, workflows — registers with an Ed25519 keypair, streams structured events, and gets real-time dashboards. No passwords. No OAuth. Just cryptographic identity and data. Solo or as a fleet.

How it works

Four steps to agent-native telemetry.

01

Generate an Ed25519 keypair

Every agent carries its own identity. No central user database, no password resets. Generate keys in a TPM, secure enclave, or environment variable. You own the keys.

const { publicKey, privateKey } = crypto.generateKeyPairSync('ed25519', {
  publicKeyEncoding: { type: 'spki', format: 'pem' },
  privateKeyEncoding: { type: 'pkcs8', format: 'pem' },
});
02

Sign once, stream forever

Sign a registration challenge to prove key ownership. Get back an entity ID. Then sign a session challenge to get a short-lived bearer token. From there, fire events at full speed — no per-request crypto.

// POST /v1/session
const signature = sign(`${entityId}:${nonce}:${timestamp}`, privateKey);
// Returns: { token: "sess_...", expiresAt: "..." }
03

Discover what is happening

Query the discovery endpoint to see what event names and dimensions your agents are sending. No more guesswork when designing dashboards — let the data tell you what to visualize.

// GET /v1/analytics/discover?workspace_id=ws_ops
{
  "eventNames": [{"name":"tool.invoke","count":3200}],
  "dimensions": {"tool":["scrape","api"],"region":["us-east","eu-west"]},
  "totalEvents": 4000
}
04

Create dashboards from discovered data

Build dashboards programmatically from the discovery response, or define them manually. Workspace dashboards aggregate across all member agents. Embed the signed URL in any WebView, iframe, or headless browser.

// POST /v1/dashboards
{
  "title": "Fleet Overview",
  "workspace_id": "ws_ops",
  "widgets": [{
    "type": "timeseries_line",
    "query": { "eventName": "tool.invoke", "aggregation": "count", "groupBy": "tool" }
  }]
}
// Returns: { "signed_url": "http://.../view/dash_...?token=..." }

Features

Privacy-first telemetry. Built for scale.

🔐

Cryptographic Identity

Ed25519 keypairs instead of passwords. Agents prove identity, receive session tokens, and stream events without human-in-the-loop.

⚡

Session Tokens

Sign once to bootstrap, then use cheap bearer tokens for high-throughput streaming. Renewable and TTL-bounded.

🔍

Auto-Discovery

Ask the system what's happening. The discover endpoint tells you what events and dimensions exist so you can generate dashboards from live data.

🏢

Workspaces & Orgs

Isolate agents into workspaces. Aggregate analytics across the whole fleet. Add members with admin or member roles.

🖼

Embeddable Dashboards

Signed URLs that carry access. Drop an iframe into your admin panel. No login sessions, no cookie banners.

🚀

Open Source

Full source on GitHub. Self-host on Railway, Render, or Kubernetes in minutes. Swap SQLite for Postgres when you need to scale.
